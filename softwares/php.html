Comandos para editar arquivos:

$texto = fopen('teste',$modo); comando para abri o arquivo, modo pesquisa na tabela

feof($texto) retorna verdadeira se o ponteiro aponta para dentro do arquivo

fgets($texto,$numero_letras) Le a linha que está o ponteiro, depende de onde o modo começou

fwrite($texto,"Linha1".PHP_EOL) escreve uma linha no arquivo

fclose($texto) fecha o arquivo salvo na variavel

file_put_contects($texto,'Conteudo de todo o arquivo') altera tudo

file_get_contents($texto) Le o arquivo todo

$texto = file('arquivo') cria um array com as linhas do arquivo

copy('local de origem','local de destino')

rename('Nome antigo','nome novo')

unlink($texto) remove o arquivo

file_exist('arquivo')Verifica se o arquivo existe

is_file('arquivo') verifica se a localização é mesmo de um arquivo

mkdir('/home/dir' cria o diretorio)

rmdir('/home/dir') remove o diretorio

scandir('/home/dir') mostra tudo que tem dentro por um array

is_dir('home/dir') já sabe neh


Manipulação de strings:

strtoupper($string) deixa tudo na string maiusculo

strtolower($string) deixa tudo na string minusculo

substr($string,$começo,$fim) mostra onde cortar na string

str_pad($string,20,'*') prenche uma string com outra

str_repeat($string,4) repete a string esse numero de vezes

str_replace($a,$b,$c) Função que substitui tudo que tenha $a em $c com o valor contido em $b

$a = 'goiaba'; $b = 'abacaxi', $c = 'Gosto de goiaba'

retorno 'Gosto de abacaxi'

strlen($string) traz a quantidade de letras
