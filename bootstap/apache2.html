Use o comando "whereis" para descobrir onde estão os arquivos de instalação do apache

# whereis apache2

O retorno do comando deve ser algo como:

/usr/sbin/apache2 /etc/apache2 /usr/lib/apache2 /usr/share/apache2 /usr/share/man/man8/apache2.8.gz

Onde:

/usr/sbin/apache2 é o diretório do binário principal do apache;

/usr/lib/apache2 é a biblioteca do apache2;

/usr/share/apache são os objetos padrões do apache.

/usr/share/man/man8/apache2.8.gz é o manual do apache.

Reinicia ou para ou inicia o servidor Apache2:

# /etc/init.d/apache2 restart | stop | start

# systemctl restart | stop | start

# service apache2 restart | stop | start

Verifica o status do servidor apache2

# systemctl status apache2

Debugar um virtual host apache2

# /usr/sbin/apache2ctl -S

Para criar um virtual host precisa criar um arquivo "exemplo.conf" na pasta "/etc/apache2/sites-enabled", dica: copia o modelo do defaut.conf

arquivo "/etc/hosts" configura os hostlocais da sua maquina, da oara criar novos hosts

Pasta "/etc" fica os arquivos de configuração do linux

Pasta "/var/www/" é o padrão para htmls do servidor

Pasta "/var/log/apache2/" ficam os registros(acesso, erro)
