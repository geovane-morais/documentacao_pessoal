Programa para acesso FTP = "VSFTPD"

Reinicia ou para ou inicia o VSFTPD:

systemctl status | start | stop vsftpd

service vsftpd status | start | vsftpd

/etc/init.d/vsftpd stop | start | vsftpd

Configurações do vsftpd:

caminho "/etc/vsftpd.conf"

listen=YES #Habilita o IPV4

listen_ipv6=NO #Desabilita o IPV6

anonymous_enable=NO< #Desabilita logins anonimos/p>

local_enable= #Autorizado o login de usuarios locais

write_enable=YES #Habilita a escrita no FTP

idle_session_timeout=600 #Define tempo de login

chroot_local_user=YES #Enjaula o usuario FTP

allow_writeable_chroot=YES #Habilita a escrita pelo usuario

usermod -s /bin #IMPORTANTE redefinir diretorio raiz do usuario
